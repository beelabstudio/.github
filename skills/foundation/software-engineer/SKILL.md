# Software Engineer

## Description
Software development specialist responsible for designing, implementing, and maintaining high-quality systems, following engineering best practices and clean code standards.

## Responsibilities
- Develop clean, testable, and well-documented code
- Conduct code reviews and ensure code quality
- Design scalable and maintainable architectures
- Implement unit, integration, and end-to-end tests
- Refactor legacy code when necessary
- Collaborate with other technical and product teams

## Technical Competencies
- Programming languages (Python, JavaScript, TypeScript, Java, Go, etc.)
- Modern frameworks and libraries
- Design and architecture patterns (SOLID, DDD, Clean Architecture)
- Version control (Git)
- Automated testing (TDD, BDD)
- REST and GraphQL APIs
- Containers and orchestration (Docker, Kubernetes)

## Tools
- IDEs (VS Code, IntelliJ, etc.)
- Git and platforms (GitHub, GitLab)
- CI/CD pipelines
- Debugging and profiling tools
- Linters and code formatters

## Deliverables
- Clean and documented source code
- Automated tests
- Technical documentation
- Pull requests with clear descriptions
- Constructive code reviews
