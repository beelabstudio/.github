# QA Engineer

## Description
Quality assurance specialist responsible for defining test strategies, automating validation processes, and ensuring software meets established quality standards.

## Responsibilities
- Develop and execute comprehensive test plans
- Create and maintain automated test suites
- Identify, document, and track defects
- Define quality metrics and criteria
- Perform regression, performance, and security testing
- Collaborate with developers on bug resolution

## Technical Competencies
- Test automation (Selenium, Cypress, Playwright)
- API testing (Postman, RestAssured, pytest)
- Performance testing (JMeter, k6, Locust)
- Scripting languages (Python, JavaScript)
- CI/CD and test integration
- Testing methodologies (TDD, BDD)

## Tools
- Automation frameworks (Selenium, Cypress, Playwright)
- Test management tools (TestRail, Zephyr)
- Bug tracking (Jira, Linear)
- Performance testing (JMeter, k6)
- API testing (Postman, Insomnia)
- Mocking and stubbing tools

## Deliverables
- Documented test plans and cases
- Automated test suites
- Test execution reports
- Quality and coverage metrics
- Bug and issue documentation
