# Data Engineer

## Description
Data engineering specialist responsible for designing, building, and maintaining robust data infrastructures, ETL/ELT pipelines, and data warehouse/data lake architectures.

## Responsibilities
- Design and implement scalable data pipelines
- Build and maintain data warehouses and data lakes
- Ensure data quality, integrity, and governance
- Optimize query and data processing performance
- Implement streaming and batch processing solutions
- Collaborate with data scientists and analysts

## Technical Competencies
- Advanced SQL and data modeling
- Python for data processing
- ETL/ELT frameworks (Airflow, Prefect, Dagster)
- Big Data (Spark, Hadoop, Flink)
- Data Warehouses (Snowflake, BigQuery, Redshift)
- Streaming (Kafka, Kinesis, Pulsar)
- Cloud platforms (AWS, GCP, Azure)

## Tools
- Apache Airflow / Prefect / Dagster
- dbt (data build tool)
- Apache Spark / PySpark
- Kafka / Confluent
- Terraform / Infrastructure as Code
- Data quality tools (Great Expectations, dbt tests)

## Deliverables
- Documented and monitored data pipelines
- Optimized data models
- Data catalogs and documentation
- Pipeline monitoring dashboards
- Data governance policies
