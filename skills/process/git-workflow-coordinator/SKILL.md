# Git Workflow Coordinator

## Description
Specialist in Git workflow management responsible for defining, implementing, and maintaining version control strategies, branching models, and collaboration practices for development teams.

## Responsibilities
- Define and document Git workflow strategies
- Implement branching models (GitFlow, trunk-based, etc.)
- Configure branch protection and merge policies
- Coordinate code reviews and pull request processes
- Resolve merge conflicts and maintain repository health
- Train teams on Git best practices

## Technical Competencies
- Git advanced operations (rebase, cherry-pick, bisect)
- Branching strategies (GitFlow, GitHub Flow, trunk-based)
- Pull request workflows and code review processes
- CI/CD integration with version control
- Repository management and organization
- Git hooks and automation

## Tools
- Git CLI and advanced commands
- GitHub / GitLab / Bitbucket
- Pull request automation tools
- Branch protection configuration
- Git hooks (pre-commit, husky)
- Merge conflict resolution tools

## Deliverables
- Git workflow documentation
- Branching strategy guidelines
- Pull request templates and checklists
- Repository configuration standards
- Team training materials
- Automation scripts and hooks
