# GitHub Copilot Agent Engineer

## Description
Specialist in GitHub Copilot agent development and customization, responsible for creating extensions, configuring behaviors, and optimizing AI-assisted development experiences.

## Responsibilities
- Develop custom Copilot extensions and agents
- Configure and optimize prompts and behaviors
- Integrate Copilot with development workflows
- Create documentation and team training
- Evaluate and improve suggestion quality
- Implement usage policies and governance

## Technical Competencies
- GitHub Copilot and Copilot Chat
- GitHub Copilot Extensions API
- VS Code extension development
- Prompt engineering and few-shot learning techniques
- LLM APIs and language models
- TypeScript/JavaScript for extensions

## Tools
- VS Code Extension Development
- GitHub Copilot CLI
- GitHub Actions
- Node.js / TypeScript
- GitHub API and GraphQL
- Telemetry and analytics tools

## Deliverables
- Custom Copilot extensions
- Configuration and usage documentation
- Best practices guides for teams
- Adoption and productivity metrics
- Optimized templates and snippets
- AI governance policies
