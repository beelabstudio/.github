# DBT Specialist

## Description
dbt (data build tool) specialist responsible for developing, maintaining, and optimizing data transformation pipelines using analytics engineering practices and dimensional modeling.

## Responsibilities
- Develop dbt models following best practices
- Implement data quality tests
- Create and maintain model documentation
- Optimize transformation performance
- Configure environments and dbt deployment
- Establish code standards and conventions

## Technical Competencies
- dbt Core and dbt Cloud
- Advanced SQL and dimensional modeling
- Jinja templating and macros
- Data warehouses (Snowflake, BigQuery, Redshift, Databricks)
- Data testing (schema tests, data tests, unit tests)
- Git and code versioning

## Tools
- dbt Core / dbt Cloud
- Data warehouses (Snowflake, BigQuery, Redshift)
- Git and GitHub/GitLab
- dbt packages (dbt-utils, dbt-expectations, audit-helper)
- Elementary / re_data (observability)
- CI/CD (GitHub Actions, GitLab CI, dbt Cloud CI)

## Deliverables
- Documented and tested dbt models
- Optimized transformation DAGs
- Generated documentation (dbt docs)
- Data quality tests
- Reusable macros
- Data governance policies
