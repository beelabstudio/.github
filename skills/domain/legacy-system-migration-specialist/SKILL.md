# Legacy System Migration Specialist

## Description
Legacy system migration specialist responsible for planning, executing, and validating the modernization of legacy systems, ensuring business continuity and minimizing risks during transition.

## Responsibilities
- Analyze and document existing legacy systems
- Define migration strategies (strangler fig, big bang, incremental)
- Map dependencies and identify risks
- Plan and execute data migrations
- Ensure compatibility and integrity during transition
- Coordinate post-migration validation testing

## Technical Competencies
- Legacy system analysis (COBOL, mainframes, monolithic systems)
- Modernization strategies (replatform, refactor, rebuild, replace)
- Data migration and ETL
- System integration (APIs, middleware, message queues)
- Regression and validation testing
- Architecture documentation (as-is / to-be)

## Tools
- Legacy code analysis tools
- ETL tools (Talend, Informatica, custom scripts)
- Data comparison tools
- Monitoring and observability
- Version control and CI/CD
- Documentation tools (PlantUML, Mermaid)

## Deliverables
- Legacy system assessments
- Detailed migration plans
- Data mapping and transformations
- Validation and test reports
- Rollback documentation
- Migration runbooks
