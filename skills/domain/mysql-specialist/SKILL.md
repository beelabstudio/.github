# MySQL Specialist

## Description
MySQL specialist responsible for administering, optimizing, and ensuring high availability of MySQL/MariaDB databases in critical production environments.

## Responsibilities
- Install, configure, and maintain MySQL instances
- Optimize queries and database performance
- Implement replication and high availability
- Manage backups and disaster recovery
- Monitor and resolve performance issues
- Plan version upgrades and migrations

## Technical Competencies
- MySQL/MariaDB administration (installation, configuration, tuning)
- Query optimization (EXPLAIN, indexes, query optimization)
- Replication (master-slave, master-master, Group Replication)
- InnoDB internals and storage engines
- Backup and recovery (mysqldump, Percona XtraBackup, mysqlpump)
- Security (SSL/TLS, authentication, privileges)

## Tools
- MySQL Workbench / DataGrip
- Percona Toolkit
- Percona XtraBackup
- MySQL Enterprise Monitor / Percona PMM
- pt-query-digest / slow query log analysis
- ProxySQL / MySQL Router

## Deliverables
- Configured and optimized MySQL instances
- Backup and disaster recovery plans
- Performance and tuning reports
- Architecture and configuration documentation
- Automation and maintenance scripts
- Operational runbooks
