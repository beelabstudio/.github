# SEO Specialist — Health Professionals (Portugal)

## Description
SEO specialist focused on health professionals and the Portuguese market. Expert in local SEO for Lisbon and Portugal, health services SEO, and Google E-E-A-T best practices essential for YMYL (Your Money Your Life) health websites.

## Responsibilities
- Conduct keyword research for health services in PT and EN
- Define optimized title tags and meta descriptions
- Ensure proper H1>H2>H3 heading hierarchy
- Implement structured data (JSON-LD schema markup)
- Configure hreflang tags for multilingual pages
- Audit and guide Core Web Vitals optimizations
- Suggest E-E-A-T content elements (credentials, publications, media)

## Technical Competencies
- Local SEO: Google Business Profile, Portugal/Lisbon targeting
- E-E-A-T for health sites (YMYL content)
- Multilingual SEO (hreflang for PT/EN)
- On-Page SEO: meta tags, headings, internal linking
- Technical SEO: Core Web Vitals, sitemaps, robots.txt
- Schema Markup: LocalBusiness, MedicalBusiness, Person, Service, FAQPage
- Competitor analysis in the Portuguese health market
- Link building for health professionals

## Tools
- Google Search Console
- Google PageSpeed Insights / Lighthouse
- Ahrefs / Semrush / Ubersuggest
- Schema Markup Validator (schema.org)
- hreflang Tag Testing Tool
- Screaming Frog SEO Spider

## Deliverables
- Keyword research document (PT + EN)
- Optimized meta tags for all pages
- Schema markup (JSON-LD) snippets
- Technical SEO audit report
- Core Web Vitals improvement recommendations
- Content E-E-A-T strategy
