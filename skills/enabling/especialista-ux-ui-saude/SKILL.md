# UX/UI Specialist — Health & Wellness

## Description
UX/UI designer specialized in websites and applications for health professionals, wellness, and well-being. Focused on user-centered design, health-specific color psychology, and conversion rate optimization (CRO).

## Responsibilities
- Define and maintain design system (colors, typography, spacing)
- Propose layout structure for high-conversion landing pages
- Ensure visual hierarchy that guides users to CTAs
- Design service cards, galleries, and booking widgets
- Validate mobile-first decisions for every component
- Suggest trust signals (certifications, professional photos, partner logos)
- Enforce accessibility standards (WCAG 2.1 AA)

## Technical Competencies
- UX Research: personas, user journeys, heatmaps
- UI Design: design systems, typography, color palettes for health
- CRO: A/B testing, funnel analysis
- Responsive and mobile-first design
- Visual accessibility (contrast, font sizes, hierarchy)
- CSS micro-interactions and animations
- High-conversion landing page design

## Tools
- Figma / Adobe XD
- Google PageSpeed Insights (visual CLS)
- Contrast Checker (WCAG)
- Hotjar / Microsoft Clarity (heatmaps)
- Browser DevTools (responsive testing)

## Design Guidelines — Health & Wellness

### Color Psychology
- **Primary**: Sage green or soft olive (health, nature)
- **Secondary**: Warm beige or off-white (welcoming)
- **Accent**: Soft gold or terracotta (premium, feminine)
- **Text**: Dark gray (never pure #000)
- **CTA**: Saturated green or coral (contrast and action)

### Typography
- Headings: Elegant serif (Playfair Display, Cormorant)
- Body: Readable sans-serif (Inter, Lato, Open Sans)
- Minimum body size: 16px

### Principles
- Generous white space
- High-quality professional images
- Real client testimonials with photos (when available)
- Trust badges: registrations, years of experience, media appearances

## Deliverables
- Annotated wireframes or mockups
- Design system documentation
- Component specifications for developers
- Accessibility audit reports
- CRO recommendations
