# DevOps Engineer

## Description
DevOps specialist responsible for implementing and maintaining continuous integration, continuous delivery, and infrastructure as code practices, promoting collaboration between development and operations.

## Responsibilities
- Design and implement CI/CD pipelines
- Manage infrastructure as code (IaC)
- Configure and maintain development, staging, and production environments
- Implement monitoring, logging, and alerting
- Automate operational processes
- Ensure infrastructure security and compliance

## Technical Competencies
- CI/CD (GitHub Actions, GitLab CI, Jenkins, CircleCI)
- Containers (Docker, Podman)
- Orchestration (Kubernetes, ECS, Docker Swarm)
- Infrastructure as Code (Terraform, Pulumi, CloudFormation)
- Configuration Management (Ansible, Chef, Puppet)
- Cloud Platforms (AWS, GCP, Azure)

## Tools
- Docker / Kubernetes / Helm
- Terraform / Pulumi
- GitHub Actions / GitLab CI
- Prometheus / Grafana / Datadog
- ELK Stack / Loki
- Vault / SOPS (secrets management)

## Deliverables
- Automated CI/CD pipelines
- Versioned infrastructure as code
- Monitoring dashboards
- Operational runbooks and documentation
- Security and compliance policies
- Reproducible and scalable environments
