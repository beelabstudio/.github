# Documentation Writer

## Description
Technical documentation specialist responsible for creating, organizing, and maintaining clear and accessible documentation for different audiences, including developers, users, and stakeholders.

## Responsibilities
- Create and maintain up-to-date technical documentation
- Develop user guides and tutorials
- Document APIs and integrations
- Establish documentation standards and templates
- Review and improve existing documentation
- Collaborate with technical teams to extract knowledge

## Technical Competencies
- Technical writing and clear communication
- Markdown, reStructuredText, and documentation formats
- API documentation (OpenAPI/Swagger)
- Docs as Code (documentation versioning)
- Information architecture
- Technical diagramming (UML, C4, flowcharts)

## Tools
- Static site generators (MkDocs, Docusaurus, GitBook)
- Diagramming (Mermaid, PlantUML, Draw.io)
- API documentation (Swagger, Redoc, Stoplight)
- Version control (Git)
- Collaboration (Notion, Confluence)
- Grammar and style (Vale, Grammarly)

## Deliverables
- Structured technical documentation
- User guides and tutorials
- API documentation
- Information architecture
- Templates and style guides
- READMEs and getting started guides
